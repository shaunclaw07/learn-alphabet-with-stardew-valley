# AGENTS.md

Dieses Projekt hält seine Konventionen, Architektur und didaktischen Regeln in
**[`CLAUDE.md`](./CLAUDE.md)** fest.

**Alle Agenten und KI-Tools lesen und befolgen `CLAUDE.md`** als verbindliche
Quelle für dieses Repository — insbesondere:

- Struktur: pro Phase eine self-contained HTML-Datei, alles inline.
- Design-Tokens (Stardew-Style), Vorlese-Engine, `localStorage`-Keys.
- Decodability-Regeln (nur Phase-1-Buchstaben, keine Digraphen/Umlaute).
- Verifikation via `node` + Browser; Sprache Deutsch; Git nur auf Zuruf.

Bei Abweichungen oder Unklarheiten hat `CLAUDE.md` Vorrang vor Annahmen.
